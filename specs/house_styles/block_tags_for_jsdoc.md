## When applicable, use the following block tags:
- @abstract or @virtual (synonymous)
- @augments or @extends (synonymous)
	- specify which parent symbol it extends from
- @author
- @callback
- @classdesc
- @constructs
- @description
- @enum
- @event
- @function
- @instance
- @param
- @returns
- @this
- @throws
